## 3. 핵심 기능 명세

### 🔴 P0 — 즉시 해결 (앱오류 35% 대응)

#### Feature 1: 충전 세션 자동 복구 (Auto-Recovery)

**문제**: 앱 충돌·네트워크 끊김 시 충전 세션이 날아가고 결제는 이미 됨

**솔루션**:
```
충전 시작 → 앱 종료/네트워크 단절
     ↓
서버에 세션 상태 저장 (heartbeat 30초)
     ↓
앱 재실행 시 → 진행 중인 세션 자동 감지
     ↓
"충전이 진행 중입니다. 이어서 진행하시겠습니까?" 팝업
```

**기대 효과**: 앱오류 관련 부정 리뷰 50% 감소 예상

**검증 지표**:
- 세션 복구 성공률 ≥ 95%
- 세션 중단 후 재연결 시간 ≤ 10초

---

#### Feature 2: QR 스캔 실패 자동 재시도 + 수동 ID 입력

**문제**: QR 인식 오류 → 충전 시작 불가 (연결실패 0.6%, 실사용 사례 다수)

**솔루션**:
```
QR 스캔 실패 (1회)
     ↓
자동 재시도 (3초 후)
     ↓
재시도 실패 시 → "충전기 ID를 직접 입력하세요" 안내
     ↓
숫자 6자리 ID 수동 입력으로 충전 시작 가능
```

**LOCA EV 사례**: 실제 사용자가 "QR 스캔 실패 → 뽑았다 꽂기 반복" 경험 기록 (유튜브 자막 데이터 확인)

---

### 🟠 P1 — 단기 개선 (결제오류 8.2% 대응)

#### Feature 3: 오프라인 RFID 멤버십 카드 결제

**문제**: 네트워크 불안정 지역에서 앱 결제 실패 → 충전 불가

**솔루션**:
- RFID 카드 태깅으로 사전 등록 크레딧 차감
- 오프라인에서도 결제 가능 (후불 정산)
- 라오스·베트남 현지 편의점 카드 충전 연계

**참고**: PTT blueplus+(태국 1위, 4.7점)는 앱 + 카드 이중 결제 지원으로 높은 만족도 유지

---

#### Feature 4: 충전 시작 전 결제 성공 확인 화면

**문제**: "결제는 됐는데 충전이 시작 안 됨" — 결제와 충전 시작이 분리되어 혼란

**솔루션**:
```
결제 완료 ──→ [결제 확인 화면 3초 표시]
                "결제 완료 ✓ | 충전 시작 중..."
                     ↓
              충전 시작 확인 신호 수신
                     ↓
              [충전 진행 화면] kWh · 금액 실시간 표시
```

---

### 🟡 P2 — 중기 개선 (UX 전반)

#### Feature 5: 충전소 실시간 상태 지도

**문제**: 충전소위치 불만 1.8% — "충전소에 갔더니 이미 다 차있음"

**솔루션**:
- 지도에서 충전기 가용 상태 실시간 표시 (available/in-use/error)
- 이동 중 충전기 예약 기능 (LOCA EV 앱 기존 기능 강화)
- 도착 예상 시간 기반 자동 추천

---

#### Feature 6: 충전 완료 자동 알림 + 이동 유예 시간

**문제**: 충전 완료 후 차량 미이동 시 추가 과금 정책 불만

**솔루션**:
- 충전 완료 5분 전 푸시 알림
- 완료 후 10분 유예 시간 (추가 과금 없음)
- 유예 시간 초과 시 분당 과금 안내
