## 3. 핵심 기능 명세

### 🔴 P0 — 즉시 해결

#### Feature 1: 충전 세션 자동 복구 (Auto-Recovery)

**근거**: 앱오류 5,840건(58.6%) + OCPP·충전기 복합 원인 확인

**문제**: 앱 충돌·네트워크 끊김·OCPP 통신 오류 시 충전 세션이 끊기고 결제만 완료됨

**솔루션**:
```
충전 시작 → 앱 종료 / 네트워크 단절 / OCPP 오류
     ↓
서버에 세션 상태 저장 (heartbeat 30초)
     ↓
앱 재실행 시 → 진행 중인 세션 자동 감지
     ↓
"충전이 진행 중입니다. 이어서 진행하시겠습니까?" 팝업
```

**검증 지표**: 세션 복구 성공률 ≥ 95% / 재연결 시간 ≤ 10초

---

#### Feature 2: QR 스캔 실패 자동 재시도 + 수동 ID 입력

**근거**: 연결실패 121건 + 유튜브 자막 "QR 스캔 실패 → 뽑았다 꽂기 반복" 실사용 기록

**솔루션**:
```
QR 스캔 실패 → 자동 재시도 (3초) → 재실패 → 충전기 ID 6자리 수동 입력
```

---

#### Feature 3: OCPP 2.0.1 표준 준수 검증 (HW 협업)

**근거**: OCPP 이슈 뉴스 169건 / 앱오류의 상당 부분 = OCPP 통신 문제

**솔루션**:
- Costel 등 HW 제조사와 OCPP 2.0.1 준수 협약
- 충전기-앱 Handshake 오류 로그 수집 → 원인 분리
- 표준 비준수 충전기 교체 우선순위 목록 관리

**구현 주체**: HW 제조사(Costel) 협업 필수

---

### 🟠 P1 — 단기 개선

#### Feature 4: 오프라인 RFID 멤버십 카드 결제

**근거**: 결제오류 1,674건(16.8%) / PTT blueplus+는 앱+카드 이중 지원으로 만족도 우위

**솔루션**:
- RFID 카드 태깅으로 사전 등록 크레딧 차감 (네트워크 불필요)
- 라오스·베트남 현지 편의점 카드 충전 연계

---

#### Feature 5: 충전 시작 전 결제 성공 확인 화면

**근거**: "결제됐는데 충전이 시작 안 됨 — 환불도 안 됨" 반복 패턴

**솔루션**:
```
결제 완료 → [결제 확인 화면 3초] "결제 완료 ✓ | 충전 시작 중..."
         → 충전 시작 신호 수신 → [충전 진행 화면] kWh·금액 실시간 표시
```

---

#### Feature 6: 외국인 여행자 다국어 UI + 국제 결제 지원

**근거**: 영어 리뷰 Negative **65.8%** (전체 평균 53.5% 대비 12.3%p 높음)
/ 영어 리뷰 빈출 단어: "payment failed", "no foreign card", "no English"

**솔루션**:
- 앱 내 영어·한국어·태국어 다국어 지원
- Visa/Mastercard 해외 카드 직접 등록 및 결제
- 여행자 1회성 QR 결제 (앱 가입 없이 충전 가능)

---

#### Feature 7: 충전기 상태 실시간 모니터링 대시보드 (HW 협업)

**근거**: 충전기결함 Negative **32%** (가장 높은 Negative 비율)

**솔루션**:
- 충전기별 실시간 상태 (정상/사용중/오류/점검중) 앱 내 표시
- 오류 충전기 자동 감지 → 운영팀 즉시 알림
- 충전기 고장 신고 기능 in-app

---

### 🟡 P2 — 중기 개선

#### Feature 8: 충전 속도 사전 표시 + 예상 소요 시간

**근거**: 충전속도 불만 1,000건(10%) / 태국어 리뷰에서 특히 높음

**솔루션**:
- 충전소 선택 시 최대 출력(kW) 및 예상 충전 시간 표시
- 완속(AC)/급속(DC) 구분 아이콘
- "현재 배터리 XX% → 80% 충전 예상 XX분" 안내

---

#### Feature 9: 충전소 실시간 상태 지도

**근거**: 충전소위치 불만 401건(4%)

**솔루션**:
- 지도에서 충전기 가용 상태 실시간 표시 (available/in-use/error)
- 이동 중 충전기 사전 예약
- 도착 예상 시간 기반 자동 추천

---

#### Feature 10: 충전기 펌웨어 원격 업데이트 (OTA)

**근거**: 충전기결함 75건 / 기기 자체 버그로 인한 서비스 중단

**솔루션**:
- Costel 등 HW 제조사와 OTA 업데이트 채널 구축
- 펌웨어 버그 패치 무중단 배포 (심야 자동 업데이트)

**구현 주체**: HW 제조사(Costel) 협업

---

#### Feature 11: 충전 완료 자동 알림 + 이동 유예 시간

**근거**: UX 전반 개선 요구

**솔루션**:
- 충전 완료 5분 전 푸시 알림
- 완료 후 10분 유예 시간 (추가 과금 없음)
