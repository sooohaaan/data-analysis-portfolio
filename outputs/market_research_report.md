## 1. 거시 환경 분석 (PEST)

### 🏛️ Political — 정치·정책

**라오스 (핵심 시장)**
- 2021년 라오스 정부 **전기차 보급 확대 국가 전략** 수립
- 수입 내연기관차 제한 정책 추진 (2030년 목표)
- EV 충전 인프라 구축에 외국 투자 허가 용이 — LOCA EV의 40개소 확장 배경
- 정부 주도 EV 충전소 설치 허가 절차 간소화 (LOCA EV 사례: 1년 내 투자금 회수)

**베트남**
- 2022년 국가 전기차 개발 전략 승인 (2030 로드맵)
- VinFast 정부 지원 기반 급성장 → Green SM 연계 생태계
- EV 보조금·세제 혜택 확대

**태국**
- 30@30 정책: 2030년까지 EV 30% 목표
- BOI 투자청 인센티브 — EV 충전 인프라 투자 지원
- 태국 KTR 기관이 EV 충전기 안전인증 체계 구축 중

**시사점**: 3개국 모두 정부 EV 육성 정책 → 규제 리스크 낮음, 정부 협력 가능성 높음

### 💻 Technology — 기술

**OCPP 표준화 — 업계 최대 과제**
- OCPP(Open Charge Point Protocol) 관련 뉴스: **169건** 수집
- Negative 비율은 낮으나 언급 빈도 1위 → 업계 전반의 화두
- 현황: 라오스 충전기 주로 GB/T(중국) + CCS2(유럽) 듀얼 포트
- **앱오류 58.6%의 상당 부분이 OCPP 통신 오류**에 기인

**충전기 하드웨어 품질**
- 충전기결함 관련 뉴스 Negative: **32%** (가장 높음)
- Costel(한국) 등 제조사 OCPP 2.0.1 준수 전환 진행 중
- OTA(원격 펌웨어 업데이트) 필요성 대두

**기회 (O)**

### 5-4. 실행 로드맵 (세부 액션 포함)

#### Phase 0 — 선행 조건 (즉시~1개월)

> KOKKOK Move 리뷰 분석 결과, EV 서비스 추가 전 **드라이버 공급 문제 해결이 선행 필요**

| 액션 | 세부 내용 | 담당 | 성공 지표 |
|------|---------|------|---------|
| 드라이버 인센티브 개편 | 배차 거부율 높은 단거리 요금 상향, 피크타임 보너스 도입 | 운영팀 | 배차 성공률 70%→90% |
| OCPP 파트너십 | Costel 등 HW 제조사와 OCPP 2.0.1 준수 협약 체결 | 사업팀 | 협약서 체결 |
| EV 드라이버 파일럿 | 비엔티안 내 EV 택시 드라이버 20명 모집·운영 | 운영팀 | 드라이버 20명 확보 |

---

#### Phase 1 — 앱 안정화 (0~3개월)

**목표**: 충전 성공률 90%+, 앱오류 58.6% → 20%

| 액션 | 세부 내용 | 근거 데이터 |
|------|---------|-----------|
| **충전 세션 자동 복구** | heartbeat 30초, 앱 재실행 시 세션 자동 감지·복구 | 앱오류 5,840건 |
| **QR 재시도 + 수동 ID** | 스캔 실패 시 3초 자동 재시도, 6자리 충전기 ID 수동 입력 | 연결실패 121건 |
| **OCPP Handshake 오류 로그** | 앱↔기기 통신 오류 로그 수집, 원인별 분류 | OCPP 이슈 169건 |
| **결제 확인 화면** | 결제 완료 후 "충전 시작 중..." 3초 확인 화면 표시 | 결제오류 1,674건 |
| **충전기 실시간 상태** | 지도에서 가용(녹색)/사용중(주황)/오류(빨강) 표시 | 충전소위치 401건 |

---

#### Phase 2 — 결제 다각화 (3~6개월)

**목표**: 결제오류 16.8% → 5%, 외국인 Negative 65.8% → 45%

| 액션 | 세부 내용 | 근거 데이터 |
|------|---------|-----------|
| **RFID 오프라인 결제** | 카드 태깅으로 사전 크레딧 차감, 네트워크 불필요 | PTT blueplus+ 벤치마크 |
| **국제 카드 지원** | Visa·Mastercard 해외 카드 직접 등록 | 영어 리뷰 65.8% Negative |
| **앱 없이 QR 결제** | 앱 가입 없이 QR 스캔 → 1회 결제 (여행자용) | 외국인 "No app needed" 요구 |
| **다국어 UI** | 영어·한국어·태국어 추가 (현재 라오어·베트남어만) | 영어 리뷰 Negative 12.3%p 높음 |
| **충전 완료 알림** | 완료 5분 전 푸시 + 10분 유예 기간 (추가 과금 없음) | UX 개선 요구 |

---

#### Phase 3 — Kokkok 슈퍼앱 통합 (6~12개월)

**목표**: Kokkok Move 앱 내 EV 충전 탭 출시, 별점 3.1점 달성

| 액션 | 세부 내용 | 기대 효과 |
|------|---------|---------|
| **Kokkok Move 앱 내 EV 탭** | 기존 택시 앱에 "EV 충전" 탭 추가 (LOCA 모델) | 기존 사용자 자연 유입 |
| **EV 드라이버 충전 할인** | Kokkok Move 드라이버 충전 요금 20% 할인 | 드라이버 충성도 확보 |
| **충전소 예약 기능** | 이동 중 충전기 사전 예약 (도착 예상시간 기반) | 충전소위치 불만 해소 |
| **파트너 충전소 확대** | 비엔티안 주요 쇼핑몰·호텔 충전소 협약 | 인프라 확장 |
| **데이터 기반 운영** | 충전 패턴 분석 → 인기 시간대·위치 최적화 | 운영 효율화 |
