## 4. 앱 ↔ 하드웨어 연계 분석

### 핵심 발견: 앱오류 35%의 복합 원인

```
[앱 리뷰에서 관찰된 증상]          [하드웨어 뉴스에서 확인된 원인]

앱오류 35%          ←──────────── OCPP·호환문제 169건 (표준 미준수)
  (5,840건)                         충전기결함 75건 (펌웨어 버그)

결제오류 8.2%       ←──────────── OCPP 결제 트랜잭션 통신 오류
  (1,266건)

연결실패 0.6%       ←──────────── 충전기↔앱 Handshake 오류
  (121건)
```

**결론**: 앱 리뷰의 "앱오류 35%"는 앱 코드 버그 단독이 아니라,
**OCPP 통신 표준 미준수 + 충전기 펌웨어 불안정성**의 복합적 원인에 기인한다.
따라서 앱 개발 팀만의 수정이 아닌 **HW 제조사 협업**이 필수적이다.

## 5. 핵심 인사이트 요약

### 🔴 즉시 해결 필요 (Critical)

1. **앱오류 35% = OCPP 통신 + 충전기 펌웨어 복합 문제**
   - 근거: 앱오류 5,840건 + OCPP 뉴스 169건 + 충전기결함 Negative 32%
   - 해결: 충전 세션 자동 복구 (앱) + OCPP 표준 준수 검증 (HW)

2. **결제오류 8.2% — 태국 1위(PTT)와의 핵심 격차**
   - 근거: 결제오류 1,266건 / PTT는 앱+카드 이중 결제로 만족도 우위
   - 해결: RFID 오프라인 결제 도입

3. **PEA VOLTA Negative 67.9% — 동남아 최악의 UX**
   - 근거: 499건 중 339건 부정
   - 인사이트: Kokkok EV가 PEA VOLTA 수준만 피해도 시장 경쟁력 확보

### 🟠 단기 개선 (High)

4. **영어 리뷰 Negative 65.8% — 외국인 여행자 이탈**
   - 근거: 영어 리뷰 평균 2.55점, 결제 수단 오류 집중
   - 해결: 국제 결제(Visa/Mastercard) + 다국어 UI 지원

5. **충전기결함 Negative 32% — 하드웨어 품질 문제**
   - 근거: 태국이 인프라 구축 완료 후 현재 품질 문제 단계
   - 해결: 충전기 OTA 펌웨어 업데이트, 실시간 기기 상태 모니터링

### 🟡 중기 개선 (Medium)

6. **충전속도 불만 10% — 태국어 리뷰 특히 높음**
   - 근거: 충전속도 관련 키워드 1,000건
   - 해결: 충전 속도 사전 표시, 급속·완속 구분 안내

7. **OCPP 표준화 169건 — 업계 화두**
   - 근거: 글로벌 OCPP 2.0.1 전환 진행 중
   - 해결: Costel 등 제조사와 OCPP 2.0.1 준수 협약
