# 📋 라오스 통계청(LSB) 데이터 수집 계획

> **작성일**: 2026-06-13 | **목적**: 라오스 현지 정량 근거 보강 — 시장 분석(TAM/SAM/SOM)·PEST의 약점이던 "라오스 1차 데이터 부족"을 공식 통계로 채움
> **신뢰도 원칙**: 공식 1차/IGO 자료만 사용 (LSB·ASEANstats·World Bank·ITU). 시장조사업체 추정치 제외.

## 1. LSB가 보유한 자료 (확인 결과)

라오스 통계청은 **통계연감(Statistical Yearbook 2024, 최신)** 과 **LAOSIS 온라인 DB**(`laosis.lsb.gov.la`, 24개 분야), **가구소비·지출조사(LECS7, 2024/25)** 를 발행한다.

| 자료 | 보유 | 출처/접근 |
|------|------|----------|
| 등록 차량(유형별: 이륜차·승용·트럭) | ✅ | LSB 연감 *Transport & Communication* → ASEANstats 미러 |
| 이동통신·인터넷 보급 | ✅ | LSB 연감 ICT · LECS7 · ITU |
| 가구 소비·지출(교통·연료 포함) | ✅ | LECS7 2024/25 |
| 인구·가구(도시별) | ✅ | 인구센서스 2015 + 연감 추계 |
| GDP·1인당 소득 | ✅ | LSB 국민계정 · World Bank |
| **EV 전용 등록대수** | ⚠️ 불확실 | 연감 EV 분리 여부 불명 → Customs 수치가 정확 |
| **EV 충전소·충전량** | ❌ 없음 | 국가통계 대상 아님 |
| **라이드헤일링·모빌리티앱 이용** | ❌ 없음 | 정부통계 아님 |

## 3. 수집 불가 항목 — 대체 출처 유지 (확정)

아래 항목은 **LSB(국가통계)에 존재하지 않으므로**, 기존 대체 출처를 그대로 유지한다.

| 불가 항목 | 사유 | 대체 출처 (유지) |
|----------|------|----------------|
| **EV 충전소 수·충전량** | 국가통계 대상 아님 (민간 인프라) | **LOCA 공식 발표** (40개소→2026년 100개소) |
| **EV 전용 등록대수** | LSB 연감 EV 분리 불확실 | **라오스 Customs 4,437대**(2024.10, Laotian Times 인용) |
| **모빌리티/충전 앱 이용** | 정부통계 아님 | **본 프로젝트 VOC** (앱리뷰 28,890 · 유튜브 · 슈퍼앱 리뷰) |

> 📌 즉, **EV 충전소·EV 등록·앱 이용 = LSB에 없음 → Customs(4,437대)·LOCA·VOC로 유지**. (README에도 명시)

## 4. 접근 시 주의

- **LAOSIS(`laosis.lsb.gov.la`)**: SSL 인증서 문제로 자동 수집 불가 → 브라우저 수동 접속 또는 통계연감 2024 PDF 다운로드.
- **ASEANstats(`data.aseanstats.org`)**: LSB 차량 데이터를 미러링하며 **REST API 제공** → 프로그래밍 수집에 가장 적합 (notebook 07에서 사용).
- **World Bank / ITU**: GDP·인구·ICT는 IGO 포털에서 안정적으로 접근 가능.
